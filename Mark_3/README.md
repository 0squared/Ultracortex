###PRINT SETTINGS

FRONT & BACK

* Material: PLA
* Supports: YES
* Raft: hopefully NO (but if supports aren't sticking, try the raft)
* Infill: 20%
* Layer Heigh: 0.2mm
* Number of Shells: 2
* Speed while extruding: 50%
